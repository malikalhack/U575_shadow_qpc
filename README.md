# Blink Project on QP/C framework
This repository contains a simple example, **Blink**, demonstrating how to toggle an LED using the _**QP/C**_ real-time framework.

## Requirements
- Supported microcontroller board is NUCLEO-U575ZI-Q
- AC6 compiler
- QP/C source
- CMake/Makefile or IDE project setup

## Getting Started 

### Clone the Repository
```bash
git clone https://github.com/malikalhack/U575_shadow_qpc.git
cd U575_shadow_qpc
```
